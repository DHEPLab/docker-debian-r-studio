# Docker Debian R Studio

## Development Requirements

+ Task file
+ Docker

## Building Container

1. Build container

```
task build
```

## To run locally

1. Spin up environment

```
task up
```

2. Once the environment is running, open browser and go to `localhost:8787`

3. Input username and password `rstudio`

4. When finished tear down environment

```
task down
```
